Comment coller le badge sur votre site

  • WordPress : ajoutez un bloc « HTML personnalisé » et collez le code dedans.
  • Wix / Squarespace : insérez un élément « Code HTML » (ou « Embed ») et collez le code.
  • Site sur mesure : collez le code directement dans votre HTML, à l'endroit voulu.
  • Aucun script, aucun suivi : ces badges sont du HTML pur. Ils ne ralentissent pas votre site et ne collectent aucune donnée sur vos visiteurs.
