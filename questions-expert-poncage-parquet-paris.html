Comment ça fonctionne : les questions sont posées via GitHub Discussions — vous devez avoir un compte GitHub (gratuit) pour poser une question. François Gaillard répond directement sur cette page. Toutes les questions et réponses sont visibles par tous et datées.

Questions essentielles : Utilisez-vous une machine planétaire ou tambour ? Quel vernis appliquez-vous et combien de couches ? L'égrenage entre couches est-il inclus ? Intervenez-vous seul ou sous-traitez-vous ? Puis-je avoir des références récentes ?
Un bon devis mentionne : le type de machine (planétaire = mieux), la marque et le nombre de couches de vernis (Bona Mega Evo 3 couches = référence), l'égrenage (souvent omis), et un tarif fixe sans surprises. Méfiez-vous des devis < 40 €/m².
