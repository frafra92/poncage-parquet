## 🚀 Procédure d'upload GitHub

### Étape 1 : Accédez à votre repo GitHub

Repo : `https://github.com/frafra92/frafra92.github.io/` (ou `poncage-parquet`)

### Étape 2 : Uploadez les fichiers HTML et XML

**Option A** : Via l'interface GitHub (web)

1. Allez dans votre repo → dossier racine
2. Cliquez "Add file" → "Upload files"
3. Sélectionnez :
   - `chateau-bazens-poncage-parquet-agen.html`
   - `sitemap.xml` (remplace l'ancienne version)
4. Message commit : `"Add Château de Bazens page + update sitemap"`
5. Commit

**Option B** : Via Git (terminal/IDE)

```bash
git clone https://github.com/frafra92/frafra92.github.io.git
cd frafra92.github.io

# Copier les fichiers ici
cp chateau-bazens-poncage-parquet-agen.html .
```
